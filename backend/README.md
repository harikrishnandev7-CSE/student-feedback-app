# Backend — Node.js & Express

This is the backend server for the Student Feedback Application.

## Files

| File | Purpose |
|------|---------|
| `server.js` | Main server file — creates Express app, defines routes |
| `package.json` | Lists project name, scripts, and dependencies |
| `node_modules/` | Auto-generated by npm install — do NOT commit this |

## API Endpoints

| Method | Route | Description |
|--------|-------|-------------|
| POST | `/feedback` | Submit a new feedback |
| GET | `/feedback` | Get all submitted feedback |

## How to Run

```bash
cd backend
npm install
npm start
```

Server runs on: http://localhost:5000
